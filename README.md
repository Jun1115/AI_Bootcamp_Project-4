# 서울시 공유자전거 '따릉이' 데이터 분석 프로젝트

-  원하는 주제를 선택하여 그에 해당하는 데이터를 수집해 팀원들과 함께 다음의 목표들을 해결하는 팀 프로젝트

## 1. 팀 내 역할 (4명)
 - 직책 : 팀장
 - 데이터 전처리 및 결과 도출
 - 근거 자료(시장 현황, 원인 분석, 해결 방안 등) 조사
 - PPT 제작 및 발표
